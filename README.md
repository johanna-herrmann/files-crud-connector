# files-crud-connectot

![logo](./logo.png)

Connector/Client for REST file storage [files-crud](https://files-crud.johanna-dev.de/). \
Written in Typescript.

## Description
This package offers simple functions to use the API of files-crud,
which highly simplifies its usage.

## Install
```sh
npm install files-crud-connector
```
